# Запуск проекта через Docker для тестирования

## Как запустить с Docker

### 1. Подготовьте `.env` файл

Создайте файл `.env` в корне проекта и добавьте в него ваш ngrok токен.  
Его можно получить в [личном кабинете ngrok](https://dashboard.ngrok.com/get-started/setup/windows).

```bash
NGROK_AUTHTOKEN=<вставьте_сюда_ваш_ngrok_токен>
```

> ⚠️ **Важно:** для корректной работы ngrok обязательно включите VPN,  

---

### 2. Соберите и запустите контейнеры

Выполните команды в терминале (находясь в папке проекта):

```bash
# Собрать образы и запустить контейнеры
docker-compose up -d --build
```
